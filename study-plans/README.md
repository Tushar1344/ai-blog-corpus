# Study plans

Focused reading lists assembled from the corpus for specific goals — interview prep, onboarding to a topic, catching up on a lab's work, etc.

Each plan links directly into the canonical entries under `../by-focus-area/` and `../by-company/` so reading lists stay in sync as the corpus updates.

## Plans

- [**Anthropic — Applied AI Engineer**](anthropic-applied-ai-engineer.md) — ~30 hrs full plan + 6-8 hr crash path. Agent/harness engineering, context engineering, evals, Anthropic's alignment culture, competitive landscape.

## How to build your own

1. Filter the [Google Sheet](https://docs.google.com/spreadsheets/d/1td1T6wCEnw4EeG05I_hH1dK1aNkLR5HJZZ-4jr-Du9A/edit) by the combination of `company` / `country` / `focus_area` / `subcategory` / `signal_overall` that matches your goal.
2. Sort by `signal_priority` descending.
3. Export to a new markdown file in this folder, grouped by tier (must-read → context → depth).
4. Add interview signal tips or discussion prompts if it's a prep plan.

## Template

```markdown
# {Goal} — study plan

One-line framing of why this plan exists.

**Scope:** estimated reading time · focus areas covered · plan structure.

## Tier 1 — Must read first
(table of 8-15 posts with priority + reason)

## Tier 2 — Context
(broader reading to round out the picture)

## Tier 3 — Competitive landscape
(what adjacent orgs are doing — usually the interview edge)

## Interview signal tips / discussion prompts
(4-6 takes or debates worth having a position on)

## Crash path (3-8 hrs)
(a 10-12 post subset for the time-constrained)

## One-click access in the corpus
(Sheet filter or direct markdown link)
```
